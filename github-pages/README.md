# Middle East Oil & Gas Jobs

A self-contained, interactive job board showcasing 613+ job opportunities across 22 leading oil and gas companies in the Middle East.

**Live site:** [View Job Board](https://YOUR_USERNAME.github.io/me-oil-gas-jobs/)

## Features

- Search across job titles, companies, locations, and categories
- Filter by Company (22), Country (8), or Category (52)
- Sort by Company, Title, or Country
- Pagination (25 jobs per page)
- Active filter tags with one-click removal
- Direct "Apply" links to official career portals
- Fully responsive design (mobile + desktop)
- Zero external dependencies — works offline

## How to Publish on GitHub Pages

1. Create a new repository on GitHub (e.g., `me-oil-gas-jobs`)
2. Upload the `index.html` file to the repository
3. Go to **Settings** > **Pages**
4. Under "Source", select **Deploy from a branch**
5. Select the **main** branch and **/ (root)** folder
6. Click **Save**
7. Your site will be live at `https://YOUR_USERNAME.github.io/me-oil-gas-jobs/` within a minute
